# VinuChain Staking Hub

A decentralized staking platform for VinuChain, allowing users to create, stake, and manage ERC20 token pools with advanced analytics and reward calculations.

## Features
- **Create Pools**: Deploy staking pools with custom APR, lock periods, and minimum stakes.
- **Stake/Unstake**: Stake tokens and withdraw with lock period enforcement.
- **Reward Calculator**: Estimate staking rewards based on amount, APR, and duration.
- **Analytics**: View TVL, average APR, total stakers, and user-specific data.
- **Contract Verification**: Copy Solidity code for verification on VinuChain Explorer.
